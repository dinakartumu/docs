# Welcome to your docs

This site is synced from your GitHub repository by [Pageloop](https://pageloop.ai).
Edit any page in the repo — or in the Pageloop editor — and publish to update the site.

## How this works

- Pages are `.mdx` (or `.md`) files in this repository.
- `docs.json` controls the site name, theme, and navigation.
- Every push to the default branch syncs automatically.

Head to the [Quickstart](/quickstart) to make your first change.
