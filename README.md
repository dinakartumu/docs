# Docs

Documentation site powered by [Pageloop](https://pageloop.ai), created from
[pageloop-docs-template](https://github.com/pageloop/pageloop-docs-template).

- `docs.json` — site name, theme, and navigation ([schema](https://docs.pageloop.ai/schema/v1.json))
- `*.mdx` — pages; file path (minus extension) becomes the URL
- Pushes to the default branch sync to your hosted docs automatically

This README is not published as a page.
